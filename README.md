# VoiceChat
## 1:N 다중 음성 채팅 프로그램
### CUK 객체패러다임 팀프로젝트
![Build Status](https://img.shields.io/badge/-JAVA-F26D85)


## Function
- Login/Logout
- Change Nickname on real time
- Kick out
- Voice volume control
- 1:N multi voice program

## Usage
If you want to play with this code, import the project into Netbeans
